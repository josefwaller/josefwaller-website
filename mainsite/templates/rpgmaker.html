Quick Tutorial

There are three main parts which each contribute to the game, as well as tester.

Sprites

This is where you can change your game's art. The animations are made of sprites, and flips back and forth to create an animation. For example, when the player runs down the screen, it will switch between the player's runDownOne and runDownTwo sprites. You can only use up to eight different colors, so try to reuse colors whenever you can.

Dialog

Here you can change the dialog between the player and each NPC. There are three different NPCs, and each one has their own conversation with the player. When talking, the player and NPC's dialog sprite will be used in the dialog box.

Level Editor

Here you change the layout of the level. You can add and remove elements or areas and change the background. Before you can test your level, you will need a player to exist for the user to control.

Tester

You can test your level in the maker, rather than going to the /rpgplayer page. Use to move, Q to talk to NPCs and to switch weapons, and space to use the current weapon.

Saving and Loading

When saving, you will need to give a password if you have not already, and then will be given an ID. The password is only needed for editing the game, so the ID can be given out to people who want to play your game without fear of them editing it. Remember both though!
